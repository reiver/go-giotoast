# go-giotoast

Package **giotoast** provides a UI **toast** components for the **Gio UI** library, for the Go programming-language (golang).

## Documentation

Online documentation, which includes examples, can be found at: http://godoc.org/github.com/reiver/go-giotoast

[![GoDoc](https://godoc.org/github.com/reiver/go-giotoast?status.svg)](https://godoc.org/github.com/reiver/go-giotoast)

## Import

To import package **giotoast** use `import` code like the follownig:
```
import "github.com/reiver/go-giotoast"
```

## Installation

To install package **giotoast** do the following:
```
GOPROXY=direct go get github.com/reiver/go-giotoast
```
